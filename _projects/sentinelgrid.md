End-to-end cyber defense data platform that captures real attacker activity with honeypots, analyzes behavior with ML, and turns it into real-time backend decisions and frontend threat visibility.
- Built a full ML workflow for honeypot data - preprocessing, feature extraction, behavioral profiling, anomaly/clustering analysis with KMeans & Isolation Forest, and output generation. 
- Created a deployment recommendation engine that converts recent session behavior into normalized honeypot-type distributions for dynamic resource allocation.
- Integrated ML into backend APIs so the system can serve current distributions, trigger on-demand pipeline refreshes, and support user entered configurations.
